1.C语言简介
	C语言是一种通用的、面向过程式的计算机程序设计语言。

2.C语言常识
	C 语言是为了编写 UNIX 操作系统而被发明的。
	C 语言是以 B 语言为基础的，B 语言大概是在 1970 年被引进的。
	C 语言标准是于 1988 年由美国国家标准协会（ANSI，全称 American National Standard 	Institute）制定的。
	截至 1973 年，UNIX 操作系统完全使用 C 语言编写。
	目前，C 语言是最广泛使用的系统程序设计语言。
	大多数先进的软件都是使用 C 语言实现的。
	当今最流行的 Linux 操作系统和 RDBMS（Relational Database Management System：关系数据库管理系统） 
	MySQL 都是使用 C 语言编写的。
	
3.为什么要使用C
	由于 C 语言所产生的代码运行速度与汇编语言编写的代码运行速度几乎一样，所以采用 C 语言作为系统开发语言。
	应用领域:
		操作系统、语言编译器、汇编器、文本编辑器、打印机、网络驱动器、现代程序、数据库、语言解释器、实体
		工具。

4.C语言特点
	简洁紧凑、灵活方便(32关键字，9种控制语句)
	运算符丰富(34个运算符)
	数据结构丰富(整型、实型、字符型、数组类型、指针类型、结构体类型、共用体类型)
	结构式语言
	语法限制不太严格，程序设计自由度大
	允许直接访问物理地址，可以直接对硬件进行操作
	程序生成代码质量高，程序执行效率高
	适用范围广，可移植性好
	C语言是处于汇编语言与高级语言之间的一种中间型程序设计语言，常被称为中级语言

5.C语言源程序
	命令即指令，也称为程序语句，是组成C语言程序的基本条件
	程序是一堆符合语法规则的指令的集合
	软件是一系列按照特定顺序组织的计算机数据和指令的集合
	结构特点:
		一个C语言源程序可以由一个或多个源文件组成
		每个源文件可由一个或多个函数组成
		一个源程序不论由多少个文件组成，有且只有一个main函数，即主函数
		源程序中可以有预处理命令
		每条程序语句都必须以分号结尾
		标识符，关键字之间至少加一个空格以示间隔
	
6.程序注释
	含义:
		用于提供给程序的使用者与维护者的程序代码	

	方式:
		1.单行注释
			以"//"作为开头，到行尾结束 
		2.多行注释
			以"/*"作为开头，"*/"作为结束
